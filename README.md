# Do Khac Minh Tam Portfolio

Personal portfolio website for introducing my profile, hobbies, hometown, future goals, resume, and contact information.

## Live Demo

This site is designed to be deployed with GitHub Pages.

After enabling GitHub Pages, the website will be available at:

```text
https://adamtam321.github.io/Portfolio/
```

## Features

- Responsive personal portfolio layout
- Video background on the home section
- Typing animation for profile text
- Hobby carousel
- Resume PDF download
- Contact information section
- Social media links

## Tech Stack

- HTML5
- CSS3
- JavaScript
- jQuery
- Typed.js
- Owl Carousel
- Font Awesome

## Run Locally

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Deploy With GitHub Pages

1. Push this project to GitHub.
2. Open the repository settings.
3. Go to `Pages`.
4. Select `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save the settings.
